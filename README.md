1. Relatório Detalhado e Escopo  

O objetivo deste projeto é realizar uma análise de dados profunda sobre os microdados de acidentes de trânsito no Estado do Espírito Santo entre os anos de 2020 e 2025. O projeto focará na identificação de correlações estatísticas entre variáveis como tipo de via, condições climáticas e gravidade dos acidentes, utilizando técnicas avançadas de Ciência de Dados para gerar insights que auxiliem na segurança viária. 


2. Justificativa do Tema  

A análise de acidentes de trânsito é um problema real de alto impacto social e urbano. A utilização de modelos computacionais para entender padrões de acidentalidade permite propor soluções inovadoras baseadas em evidências, indo além da simples observação e permitindo uma compreensão técnica de fenômenos que afetam a saúde e a mobilidade pública. 


3. Sociedade Impactada  

A sociedade beneficiada por este projeto compreende: 

Ciclistas e Pedestres: Que compõem o grupo mais vulnerável no trânsito urbano. 

Gestores Públicos: Que podem utilizar as métricas e testes estatísticos para planejar intervenções de infraestrutura. 


4. Metodologia Científica  

Como requisito do curso de Ciência da Computação, a metodologia seguirá as etapas: 

Coleta de Dados: Uso de bases públicas (DETRAN-ES/Dados Governamentais). 

Análise Estatística: Aplicação de testes estatísticos para validar hipóteses sobre os acidentes. 

Modelagem: Implementação de modelos de aprendizado de máquina (Machine Learning) para classificação ou predição de riscos. 

Tecnologias: Python para processamento, SQL para manipulação de grandes volumes e GitHub para versionamento.
